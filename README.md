# vue-app-start

## Principles

- Use Vue + ElementUI + TypeScript + JSX
- Do not use .vue files
- Do not use Vue templates

## Project setup
```sh
npm install
```

### Compiles and hot-reloads for development
```sh
npm run serve
```

### Compiles and minifies for production
```sh
npm run build
```

### Run your unit tests
```sh
npm run test:unit
```

### Lints files
```sh
npm run lint
```
